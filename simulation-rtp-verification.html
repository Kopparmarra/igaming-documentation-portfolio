Method

The simulation uses the same simplified logic as the sample game:

  • Three reels.
  • Each reel draws one symbol from the weighted symbol pool.
  • Three matching symbols pay the defined multiplier.
  • The first two matching symbols return the bet as a push.
  • All other outcomes return zero.
if reel1 == reel2 == reel3:
    payout = bet × symbol_multiplier
elif reel1 == reel2:
    payout = bet
else:
    payout = 0

The simulation is an illustrative verification tool. It does not replace a certified RTP calculation tied to the exact production game model.
