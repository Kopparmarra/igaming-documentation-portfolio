Payout Logic / RTP Calculation

Calculation method for the sample game’s theoretical weighted-pool RTP.

Model Symbols Formula Calculation Result Certification note

Model

This calculation treats each reel as an independent draw from the same weighted symbol pool. Total symbol weight is 110. Payouts are expressed as bet multipliers, so bet size does not change RTP.

Model scope: The RTP calculation and simulation use the weighted-pool model and payout rules from the sample game. The embedded playable prototype may show a slightly different exact strip RTP because it generates 80-position reel strips on page load.

Formula

p = symbol weight / total symbol weight

Three-of-a-kind contribution:
p × p × p × payout_multiplier

Push contribution:
p × p × (1 - p) × 1

Total expected return:
sum(three-of-a-kind contributions) + sum(push contributions)

RTP:
total expected return × 100

Certification note

The RTP value must be traceable to the exact game model being certified. If the product uses fixed reel strips, the RTP must be calculated from those strips. If the product uses weighted outcome generation, the certified model must define the weighting and selection process.

The embedded prototype builds random 80-position strips on page load and guarantees at least one occurrence of each symbol. Therefore, the exact strip RTP shown in the prototype can differ slightly from the weighted-pool RTP above.
