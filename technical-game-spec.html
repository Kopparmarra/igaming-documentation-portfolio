Technical Game Specification

Internal behavior specification for the sample slot.

State flow

StateExpected behavior
IdlePlayer can select bet and start spin.
Spin startBet is deducted, previous result cleared, spin button disabled.
Reels spinningEach reel animates to a target position.
EvaluationCenter-row symbols are read after all reels stop.
ResultPayout is applied once and result message is shown.

Reels

The evaluation row is the center visible row. Reel stop order is sequential. Visual result highlighting must match the evaluated symbols.

Balance

Balance is changed in two controlled steps: deduction at spin start and payout addition after result evaluation.

Evaluation

if reel1 == reel2 == reel3:
    payout = bet × symbol_multiplier
elif reel1 == reel2:
    payout = bet
else:
    payout = 0
